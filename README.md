# Pixiv-Nginx-GUI
## 📝 简介
这是按照 https://2heng.xin/2017/09/19/pixiv/ 中的步骤制作的 [Pixiv-Nginx](https://github.com/mashirozx/Pixiv-Nginx) 的**自动化配置**程序，
并同时实现了**可视的 Nginx 管理**。

文件下载通过 https://git.moezx.cc/mirrors/Pixiv-Nginx 镜像或加速代理实现，**可能会出现超时或失败的情况**，重试即可。
